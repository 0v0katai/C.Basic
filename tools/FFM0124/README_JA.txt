fx-CG 10/20/50 専用 C.Basic フォントエディタ　FEINT
	Font File Manager (FFM) 用サブルーチン（予定）　Version 1.24　マニュアル
									by Colon

===============================================================================
Font File Manager について
===============================================================================
このプログラムは、sentaro 様によるアドイン、C.Basic for CG のためのものです。
この C.Basic for CG を用いて、ツル様、sentaro 様により、外部フォント作成ツール "FEINT" が作成されました。
当プログラムは、C.Basic for CG および FEINT の更なる有効活用のために作成しております。
現在は単体でのみ動作しますが、将来的には FEINT のサブルーチンとして動作する予定です。

===============================================================================
動作環境
===============================================================================
fx-CG 10/20/50、Graph 90+E で、C.Basic for CG ver 1.44β以降をお使いください。
また、セットアップの "HeapRAM Size" を 118 KB 以下に設定してください。
HeapRAM Size が 119 KB 以上の場合でも一応動作しますが、画面の背景画像が描画できなくなります。
Colon 作成の日本語アドインにより、一部日本語で動作しますが、完全ではありません。

===============================================================================
警告
===============================================================================
安全のため、'#Break0 コマンドを絶対に無効化しないでください。
ポップアップウインドウを表示中にプログラムが中断された場合、プログラムが正しく動作しなくなります。
大抵は C.Basic の再起動で元に戻りますが、最悪の場合、システムがクラッシュする恐れがあります。

誤ってソースコードを編集してしまわないよう、g3m ファイルにパスワードロックをかけています。
パスワードは "FFMColon" です。ソースコードの改変は自由ですが、自己責任で行ってください。

===============================================================================
インストールするファイル
===============================================================================
"FFM0124.g3m" を電卓のストレージメモリに置いてください。
同一のディレクトリにあるフォントファイルのみを操作できます。

また、任意で、同梱のフォントファイルもストレージメモリにおいてください。

現在このアプリケーションは、"FFM0124.g3m" と同一のフォルダ内のファイルのみを操作できます。

===============================================================================
使用方法
===============================================================================
1.ファイルリスト画面
	[1ページ目]
	　-[F1](LOAD)…ファイルをフォントデータに設定する
	　-[F2](VIEW)…ビットマップを画面に表示する (何かキーを押すと戻ります)
	　-[F4](DETAIL)…フォントファイルの詳細情報を表示する (無効なファイルの情報は表示しません)
	　-[F5](DELETE)…ファイルを削除する
	　-[F6](→)…次のページへ

	[2ページ目]
	　-[F1](SEARCH)…ファイルを検索する
	　-[F5](CHAR)…現在のフォントデータを表示する
	　-[F6](→)…前のページへ

	[1・2ページ目共通]
	　-[↑]…1つ前のファイルを選択する
	　-[↓]…1つ次のファイルを選択する

	　-[EXIT]…ファイルリスト先頭へ、または終了

	　-[SHIFT]
	　　--[EXIT](QUIT)…終了
	　　--[OPTN]…背景ON/OFF
	　　--[7](CAPTURE)…画面キャプチャー
	　　--[AC/ON](OFF)…電源オフ

2.フォントファイルの詳細情報を表示する
	ファイル名、フォントファイルの種類、ファイルヘッダの情報を表示します。
	　-[F1](LOAD)…ファイルをフォントデータに設定する
	　-[F2](VIEW)…ビットマップを画面に表示する (何かキーを押すと戻ります)
	　-[F5](DELETE)…ファイルを削除する
	　-[F6](CHAR)…現在のフォントデータを表示する
	　-[↑]…1つ前のファイルを選択する
	　-[↓]…1つ次のファイルを選択する

	　-[EXIT]…ファイルリストに戻る

	　-[SHIFT]
	　　--[EXIT](QUIT)…ファイルリストに戻る
	　　--[7](CAPTURE)…画面キャプチャー
	　　--[AC/ON](OFF)…電源オフ

3.現在のフォントデータを表示する (メッセージは英語のみ)
	[1] から [4] のキーを押すと、対応するフォントデータが表示されます。
	[EXIT] キーを押すと、ファイルリスト画面に戻ります。

	フォントデータ表示時に上下キーを押すと改ページします。
	また、[EXIT] キーを押すと前の画面に戻ります。
