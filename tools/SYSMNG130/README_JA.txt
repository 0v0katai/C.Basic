System Manager について
===============================================================================
このプログラムは、sentaro 様によるアドイン、C.Basic for CG で作成したものです。
純正の System Manager では出来ないくらい詳細な設定・管理が可能です。

動作環境
===============================================================================
C.Basic for CG Ver 1.44β build 10 以降をお使いください。
また、セットアップの "HeapRAM Size" を 118 KB 以下に設定してください。
Colon 作成の日本語アドイン Ver 0.01 以降により、日本語で動作します。

使用方法
===============================================================================
純正 System Manager と異なる部分のみ記述します。

1. システムメニュー画面
	ほぼ純正互換ですが、RESET と OS Update はありません。
	・[F1](DISPLAY)…表示画面の明るさを調整する (一部機能拡張)
	・[F2](PWRProp)…オートパワーオフ時間やバックライト点灯時間を設定する (一部機能拡張)
	・[F3](LANGUAGE)…言語設定を行う
	・[F4](VERSION)…バージョン情報を表示する
	・[F6]→[F1](BattSet)…使用する電池の種類を選択する
	・[F6]→[F2](UserName)…ユーザー名を登録する
	※[F1](DISPLAY) は、エミュレータでは利用できません。

2. 表示画面の明るさを調整する (一部機能拡張)
	純正よりも詳細な設定ができます。
	※エミュレータでは利用できません。

	・上下キーで項目を選択します。
	　"EN2" はメインバックライトレベル、"PWM DUTY" はサブバックライトレベルです。
	・メインバックライトレベル (EN2) は、LOW と HIGH の 2 段階です。
	　LOW にするには [-] または [←] を、HIGH にするには [+] または [→] を押します。
	・サブバックライトレベル (PWM DUTY) は 6 から 255 までの 250 段階です。
	・1段階低くするには [-] または [←] を、1 段階高くするには [+] または [→] を押します。
	　[EXE] を押すと、手入力モードになります。
	　　1 以上 5 以下の値は 6 に、256 以上の値は 255 に丸められます。
	　　1 未満の値を入力する、または、何も入力せずに [EXE] を再び押すと、変更前のレベルのままになります。
	　※ 6 未満の値は無効であるため、本プログラムでは設定できないようにしています。
	・ファンクションキーで、純正と同じレベルに変更できます。
	　純正と同じレベルになると、その値が示されたファンクションメニューが黒く反転します。
	　※ [F1] (0) は、電卓の自己診断機能 (イースターエッグ) のレベル 0 と同じ値です。

	純正 System Manager で設定した値自体は変更できません。
	基本的に、電源を入れなおすと純正 System Manager で設定した値に戻りますが、本プログラムの DISPLAY 設定画面で電源を OFF/ON すると、詳細設定の値は保持されます。

3. オートパワーオフ時間やバックライト点灯時間を設定する (一部機能拡張)
	純正よりも詳細な設定ができます。

	・上下キーで項目を選択します。
	・1段階低くするには [-] または [←] を、1段階高くするには [+] または [→] を押します。
	・オートパワーオフ時間の設定は、1 分から 9999 分 (1 分刻み) が有効です。
	・バックライト点灯時間の設定は、30 秒から 63 分 30 秒まで (30 秒刻み) の間で、かつ、オートパワーオフ時間以下の値です。
	　オートパワーオフ時間をバックライト点灯時間より短く設定しようとすると、自動でバックライト点灯時間がオートパワーオフ時間と同じに設定されます。
	・オートパワーオフ時間の設定時に [EXE] を押すと、手入力モードになります。
	　1 未満または 10000 以上の値を入力する、あるいは、何も入力せずに [EXE] を再び押すと、変更前の値のままになります。

	純正 System Manager で設定した値自体を変更します。純正にない値を設定すると、純正 System Manager で正しい値を表示できなくなります。


その他の操作方法は、純正 System Manager と同様です。
